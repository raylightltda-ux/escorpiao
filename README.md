# 🦂 Monitor de Escorpiões Brasil

Um agente inteligente de monitoramento em tempo real de notícias sobre acidentes com escorpiões no Brasil. Perfeito para dedetizadoras, profissionais de saúde e empresas de controle de pragas.

## 🎯 Funcionalidades

- ✅ **Busca Automática 2x/dia** (08:30 e 16:30 - Horário Brasília)
- ✅ **Banco de Dados Persistente** - Todas as notícias são salvas localmente
- ✅ **Deduplicação Automática** - Remove notícias repetidas 100%
- ✅ **Filtro de Data** - Apenas notícias de Março/2026 em diante
- ✅ **Filtros Avançados** - Por estado, cidade e gravidade
- ✅ **Exportação CSV** - Leve dados para Excel/apresentações
- ✅ **Compartilhamento WhatsApp** - Compartilhe o link em 1 clique
- ✅ **Mobile-Friendly** - Funciona perfeitamente em celulares
- ✅ **Interface Intuitiva** - Design moderno e responsivo

## 📊 Como Funciona

### 🔍 Buscando Notícias

1. **Clique em "🔍 Buscar"** para fazer uma busca manual
2. O agente procura notícias em tempo real na internet
3. Notícias são automaticamente adicionadas ao banco de dados
4. Duplicatas são removidas automaticamente

### ⏰ Agendamento Automático

O agente busca automaticamente **2x por dia**:
- 🕐 **08:30** - Primeira busca (manhã)
- 🕔 **16:30** - Segunda busca (tarde)

*Deixe a página aberta nesses horários para funcionar!*

### 🗺️ Filtros

**Por Estado:**
- Clique em qualquer estado para ver apenas os casos lá
- Mostra quantidade total de casos por estado

**Por Gravidade:**
- Leve: Casos simples com atendimento
- Moderada: Casos com complicações
- Grave: Casos críticos que necessitam urgência

**Combine filtros:**
- Selecione um estado + uma gravidade para resultados específicos

### 📥 Exportar Dados

Clique em **"📥 Exportar CSV"** para:
- Baixar todas as notícias em formato Excel
- Levar para apresentações com dedetizadoras
- Fazer análises detalhadas

## 🐛 Troubleshooting

### "Página não carrega"
- Verifique sua conexão com a internet
- Limpe o cache do navegador (Ctrl+Shift+Delete)
- Tente em outro navegador

### "Notícias não aparecem"
- Clique em "Buscar" para carregar primeira vez
- Deixe a página aberta nos horários de agendamento (08:30 e 16:30)
- Verifique a conexão com internet

### "Dados desapareceram"
- Verificar se clicou em "Limpar" acidentalmente
- Tente restaurar via histórico do navegador
- Se não exportou, infelizmente foram perdidos

### "Não consigo exportar CSV"
- Verifique se tem permissão de download no navegador
- Tente salvar em pasta diferente
- Tente em outro navegador
